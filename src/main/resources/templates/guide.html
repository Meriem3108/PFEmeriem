Guide d'Utilisation

Découvrez comment utiliser notre application de traduction étape par étape.

1. Création de Compte

Étape 1: Inscription

Accédez à la page d'inscription et remplissez le formulaire avec votre nom d'utilisateur et mot de passe.

Vous pouvez également vous connecter rapidement via Facebook ou GitHub en utilisant les boutons disponibles.

Étape 2: Connexion

Une fois le compte créé, accédez à la page de connexion. Saisissez votre identifiant et mot de passe pour vous connecter à l'application. Utilisez l'option "Se souvenir de moi" pour simplifier vos futures connexions.

2. Tableau de Bord

Étape 1: Accès au Tableau de Bord

Une fois connecté, vous accédez au tableau de bord. Ce dernier affiche des statistiques et des informations importantes concernant votre compte et vos fichiers à traduire.

Étape 2: Navigation dans le Tableau de Bord

Depuis le tableau de bord, vous pouvez gérer vos fichiers, accéder à la section de traduction, et visualiser des statistiques sur vos traductions passées.

3. Traduction de Fichiers

Étape 1: Importer un Fichier XML

Accédez à la section de traduction et importez un fichier XML de type template ou scénario en cliquant sur le bouton "Importer". Le fichier sera affiché dans une boîte de texte pour prévisualisation avant la traduction.

Étape 2: Lancer la Traduction

Après avoir importé votre fichier, cliquez sur le bouton "Traduire" pour lancer le processus de traduction via notre système. La traduction sera générée en quelques secondes.

Étape 3: Télécharger le Fichier Traduit

Une fois la traduction terminée, le fichier traduit sera affiché dans une zone de texte. Cliquez sur "Télécharger" pour enregistrer le fichier traduit sur votre appareil.

Conseils Utiles

Assurez-vous que vos fichiers XML sont bien structurés avant de les importer dans l'application pour éviter les erreurs de traduction.

Utilisez l'option "Aperçu" après l'importation pour vérifier le contenu du fichier avant de le soumettre à la traduction.

Besoin d'Aide ?

Si vous rencontrez des problèmes ou avez des questions, veuillez consulter notre page d'assistance ou contactez notre support technique.
